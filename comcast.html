Comcast Virtual Private Line

Ethernet Built For Multi-Point Connectivity

Ethernet Virtual Private Line (EVPL) provides an Ethernet Virtual Connection (EVC) between multiple customer locations, and is an ideal replacement for frame relay or ATM services. This service also supports Service Multiplexed User Network Interfaces (UNI), which enables a single physical connection to Customer Premise Equipment (CPE) for multiple virtual connections.
